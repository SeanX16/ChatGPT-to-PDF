# 版本更新日志

## 📌 v1.3.4

📅 2025-04-20

- 🚀 插入位置逻辑简化：固定插入到顶部按钮栏的最左侧

## 📌 v1.3.3

📅 2025-04-20

- 🌏 PDF 按钮文字现已支持多语言显示：
  - 简体中文：另存为PDF
  - 繁體中文：另存為PDF
  - 日语：PDFとして保存
  - 韩语：PDF로 저장
  - 其余语言默认显示为英文：Save as PDF
